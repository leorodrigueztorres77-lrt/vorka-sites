# Partner Outreach Docs

## Cuándo usar

Leo quiere acercarse a un partner que ya tiene acceso a los negocios objetivo
(contadores para restaurantes, proveedores médicos o aseguradoras para
consultorios) para que refieran clientes a cambio de comisión.

## Instrucciones

1. Identifica el tipo de partner y a qué vertical apunta.
2. Redacta un documento corto (equivalente a 1-2 páginas) con: qué es Vorka en una
   línea, qué gana el partner (comisión recurrente del 20% durante el primer año,
   el mismo esquema que usa Ordercash con sus propios canales de referido), cómo
   funciona el proceso de referido paso a paso, y datos de contacto de Leo.
3. Redacta además un mensaje corto (WhatsApp o email) para el primer acercamiento,
   tono profesional pero cercano, que invite a una llamada breve en vez de
   explicar todo por escrito de entrada.
4. Si Leo pide el documento formal en Word, usa el skill `docx` disponible en
   Claude Code. Si solo pide el mensaje de acercamiento, entrégalo como texto
   simple.

## Nota

No prometas nada de Ordercash en las propuestas dirigidas a partners del vertical
consultorio — ese producto no aplica a ese vertical todavía (ver CLAUDE.md).
