# Copywriting de Negocio

## Cuándo usar

Cuando ya se va a construir o personalizar la web de un cliente y hace falta el
texto de sus secciones.

## Instrucciones

1. Identifica el vertical del negocio (restaurante | consultorio) — el tono cambia:
   - **Restaurante:** cercano, apetitoso, invita a pedir o visitar.
   - **Consultorio:** profesional, genera confianza, resalta credenciales y trato
     humano.
2. Usa español latinoamericano neutro, sin modismos muy locales que no viajen bien
   si el negocio tiene o planea tener sucursales en otra ciudad.
3. Genera, por sección: propuesta de valor (1-2 líneas para el hero), texto de
   "sobre nosotros" o "sobre el consultorio", descripción de cada servicio o plato
   destacado, título SEO (máx. 60 caracteres) y meta descripción (máx. 160
   caracteres) por página.
4. Si el negocio mencionó diferenciadores concretos (años de experiencia,
   ingredientes locales, especialidad poco común), destácalos explícitamente — es
   lo que los distingue de la competencia genérica.
5. Este mismo skill se reutiliza para generar los textos de actualización mensual
   de contenido (ver skill `maintenance-updates`) — mantén el mismo tono ya
   establecido para ese cliente en entregas posteriores, no lo reinventes cada mes.
